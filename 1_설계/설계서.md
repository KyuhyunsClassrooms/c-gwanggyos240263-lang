## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)



**방법  구조체 사용 **
#define MAX_NAME_LENGTH 50

typedef struct{
    int id; //항목 숫자
    char subject[MAX_NAME_LENGTH]; //과목 이름을 넣을 공간
    int total_units; //전체 분량 (페이지 또는 챕터)
    int current_progress; //현재진도량
}LearningItem;

LearningItem items[100]; //최대 100개의 과목항목을 저장 가능한 배열
