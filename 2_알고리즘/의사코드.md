## 2. 핵심 기능 1: (add_item)
시작

만약 항목 수 (item_count)가 100개 이상이면:
"최대항목개수(100개)에 도달했습니다" 출력

그렇지 않으면

과목 이름( 최대 49자 띄어쓰기 허용)입력받기
과목의 전체 분량(페잊 또는 챕터)값 입력받기
현재 진도량(페이지 또는 챕터)값 입력받기

입력한 과목 이름을 구조체배열의 멤버(char subject[])에
저장
입력한 과목의 전체 분량을 구조체배열의 멤버(int total_units)에 저장
입력한 과목의 현재 진도량을 구조체 배열의 멤버(int current_progress)에 저장

과목수 (item_count)를 1증가

"항목이 성공적으로 추가되었습니다.('과목명' ID:id값)"출력

종료


## 3. 핵심 기능 2: (delete_item())
시작

만약 저장된 과목 수가 0이라면:
"삭제할 학습 항목이 없습니다."출력
함수 종료

그렇지 않으면
기존의 테이블 형식의 구조를 보여주는 함수를 호출한 다음
삭제할 항목의 ID를 입력받아 항목에 해당하는 행을 모두 삭제하고 아래에 있는 항목들을 끌어올림
전체 항목수를 하나 감소함
"항목이 성공적으로 삭제되었습니다." 출력
종료

시작

"해당 ID를 가진 항목을 찾을수 없습니다."출력후 종료

현재 진도량을 보여주고 새로운 진도량을 입력 받음
"'과목이름' 항목의 진도가(새로입력한 값)으로 업데이트되었습니다."출력

종료
