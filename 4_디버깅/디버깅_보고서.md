## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
실행이 안되요

```
프로그램이 작동되지 않는다
```

### 2) 오류가 발생한 코드
```c
 printf("6. 종료 및 저장 \n;
```

### 3) 왜 오류가 생겼나요?
괄호를 닫지 않았어요
```

### 4) 어떻게 고쳤나요?
```c
 printf("6. 종료 및 저장 \n");
```

### 5) 고친 후 결과
-잘 작동해요

## 🧪 프로그램 테스트 결과

### 테스트 1
- **무엇을 테스트했나요?** 과목명과 학습량 확인
- **어떻게 테스트했나요?** 과목명을 입력하고 총페이지 수와 진행된 페이지 수를 작성했어요
- **결과:** ✅ 성공 
### 테스트 2
- **무엇을 테스트했나요?** 이전의 입력한 과목(진도율)삭제
- **어떻게 테스트했나요?** 4(delete_item())을 입력하여 이전의 입력한 내용을 삭제 했어요
- **결과:** ✅ 성공
